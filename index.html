About This Website

This website presents visualizations for renewable energy usage, CO₂ emissions, forest area, and power consumption for Norway, Spain, Brazil, India, and Egypt from 2010 to 2024. All charts were created using Python (Altair) and exported as images.

Each page focuses on a different type of visualization: scatter plots use size and color to encode multiple attributes, bar charts compare categories, and trend charts show how indicators change over time.
